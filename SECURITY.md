# Security Policy

## Supported Versions

currently supported versions of PraktiCalc:

| Version | Supported          |
| ------- | ------------------ |
| 1.4.x   | :white_check_mark: |
| 1.3.x   | :white_check_mark: (semi-LTS) |
| 1.2.x   | :x:                |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you found a security vulnerablility, please contect me [via e-mail](mailto:karldpbkz@gmail.com). 
Please note that the Windows 7 builds are built with Python 3.8.20, which is end-of-life. That is something that I unfortunately cannot change, as there is no newer version supporting Windows 7.
